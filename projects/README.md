# Revenue-ready project portfolio

These projects are product blueprints, not revenue promises. Each starts with a recurring customer problem, defines a measurable outcome, calculates cost per successful job, and adds safeguards before automation.

| Project | Buyer | Outcome | Suggested model |
|---|---|---|---|
| Knowledge Assistant | SMEs, associations | cited answers from approved documents | setup + subscription + usage |
| Document Intelligence | accounting, operations | structured fields from PDFs and scans | per document + review tier |
| Voice Receptionist | local services | answered calls and confirmed bookings | setup + monthly minutes |
| E-commerce Catalog Engine | retailers, exporters | multilingual product pages | per SKU + subscription |
| Localization Studio | educators, media teams | transcript, translation, dubbing | per media minute |
| Property Media Studio | real estate, tourism | listing copy and visual assets | per property package |
| Support Triage | support teams | classify, retrieve, draft, escalate | per seat + usage |
| Proposal Copilot | agencies, B2B vendors | requirements matrix and draft | per workspace |
| Niche Tutor | training providers | grounded lessons and assessment | per learner/month |
| Market Intelligence | operators, analysts | cited recurring brief | subscription |

The integrated capstone, `publishing_engine`, combines document extraction, metadata generation, multilingual packaging, RAG, audio planning, and launch assets for authors and publishers.
