Open-source research software

AI APIs in Practice

A visual and executable AI API playbook for multimodal, agentic, production-ready, and commercially realistic applications.

Created by Faramarz Kowsari, this project translates modern artificial-intelligence APIs into reproducible engineering patterns, offline-testable examples, safety utilities, evaluation workflows, and practical product blueprints.

Core capabilities

Language and reasoning

Prompting, structured outputs, schema validation, reasoning workflows, and code generation.

Agents and tools

Function calling, tool use, agent orchestration, computer interaction, and MCP patterns.

RAG and knowledge

Embeddings, chunking, retrieval, reranking, vector stores, and RAG evaluation.

Multimodal AI

Vision, OCR, image generation, speech, realtime voice, and video systems.

Production engineering

Security, privacy, retries, observability, caching, routing, and cost controls.

Commercial blueprints

Product concepts based on measurable outcomes, unit economics, safeguards, and human review.
